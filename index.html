geopython

geopython is a GitHub organization comprised of Python projects related to geospatial.

  • OWSLib
  • pycsw - OGC CSW server implementation
    • workshop
  • PyWPS - OGC WPS server implementation
  • GeoHealthCheck -support monitoring OGC services uptime and availability
  • CadTools - Some tools to perform cad like functions in QGIS
  • istSOS - OGC SOS server implementation

Also join us in irc://freenode.net/#geopython.

For more geospatial projects, check out the Toblerity Project.
